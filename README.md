# 👹安装教程/使用说明

取得阿里云盘token方法如下：

先通过浏览器打开阿里云盘官网并登录网页版：https://www.aliyundrive.com/drive/ 登录成功后，

打开开发者工具 ① 点击 Console进入控制台 ② 在控制台输入 

JSON.parse(localStorage.token).refresh_token

③ 复制返回的32位字符串，不要复制双引号
